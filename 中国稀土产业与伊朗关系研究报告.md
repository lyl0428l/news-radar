## 一、中国稀土产业全景

### 1.1 什么是稀土

稀土元素（Rare Earth Elements, REE）是元素周期表中镧系15个元素加上钪（Sc）和钇（Y）共17种金属元素的统称。尽管名为"稀土"，其在地壳中的丰度并不算特别低，但由于分布分散、提取困难、加工技术门槛高，使其成为具有高度战略价值的资源。

稀土通常分为两大类：

- **轻稀土**（Light REE）：镧、铈、镨、钕等，储量相对丰富，应用广泛
- **重稀土**（Heavy REE）：钆、铽、镝、钬、铒等，储量稀少，战略价值更高

### 1.2 稀土的战略应用领域

| 应用领域 | 关键稀土元素 | 具体应用 |
|---------|------------|---------|
| 永磁材料 | 钕、镨、镝、铽 | 电动汽车电机、风力发电机、硬盘驱动器 |
| 催化剂 | 镧、铈 | 石油炼化催化裂化、汽车尾气净化 |
| 荧光/发光材料 | 铕、铽、钇 | LED照明、显示屏、医疗影像 |
| 抛光材料 | 铈 | 光学玻璃、半导体晶圆抛光 |
| 电子/通信 | 钕、钐、钆 | 5G设备、卫星通信、雷达 |
| 军事国防 | 钐、钕、镝 | 精确制导武器、战斗机引擎、声呐系统 |
| 新能源 | 镧、铈、钕 | 镍氢电池、储氢合金 |
| 玻璃/陶瓷 | 铈、镧、钕 | 特种玻璃添加剂、高性能陶瓷 |

稀土被称为"工业维生素"——用量不大，但不可或缺。在绿色能源转型（电动车、风电）和高科技竞争（芯片、5G、AI硬件）的背景下，稀土的战略重要性持续上升。

### 1.3 中国稀土的全球地位

#### 1.3.1 储量

根据美国地质调查局（USGS）数据，全球稀土氧化物（REO）储量约为1.3亿吨，主要分布如下：

| 国家 | 储量（万吨REO） | 全球占比 |
|------|----------------|---------|
| 中国 | 4,400 | 约33-35% |
| 越南 | 2,200 | 约17% |
| 巴西 | 2,100 | 约16% |
| 俄罗斯 | 2,100 | 约16% |
| 印度 | 690 | 约5% |
| 澳大利亚 | 410 | 约3% |
| 美国 | 230 | 约2% |
| 其他国家 | 约1,000+ | 约8% |

中国储量虽占全球约三分之一，但其真正的主导地位体现在开采和加工环节。

#### 1.3.2 开采与生产

- 中国稀土矿产量约占全球的 **60-65%**（约24万吨REO/年，2024年数据）
- 缅甸（实际多为中国企业在缅投资运营）已成为第二大生产来源
- 美国MP Materials（Mountain Pass矿）、澳大利亚Lynas是中国以外最大的生产商

#### 1.3.3 加工与冶炼——真正的垄断环节

这是中国最核心的优势所在：

- 中国控制着全球约 **85-90%** 的稀土分离冶炼产能
- 约 **90%** 的稀土永磁体（NdFeB磁铁）在中国生产
- 即使其他国家开采了稀土矿石，大部分仍需运到中国加工

这种"矿-冶-材-器件"全产业链的控制力，是中国稀土产业最具战略价值的部分。

### 1.4 中国稀土产业发展历程

| 时期 | 阶段 | 特点 |
|------|------|------|
| 1950-1970年代 | 起步期 | 地质勘探，基础研究（"稀土之父"徐光宪） |
| 1980-1990年代 | 扩张期 | 低价出口换外汇，"中东有石油，中国有稀土"（邓小平） |
| 2000-2010年 | 整合期 | 行业乱象整治，出口配额制度，引发中美欧WTO争端 |
| 2010-2020年 | 战略化 | 六大稀土集团整合，资源税改革，环保标准提升 |
| 2020年至今 | 博弈期 | 出口管制工具化，《稀土管理条例》出台，供应链安全博弈 |

### 1.5 中国稀土政策工具

2023年以来，中国稀土政策发生了重大变化：

1. **《稀土管理条例》**（2024年10月正式施行）：将稀土资源归国家所有，建立全链条监管体系
2. **出口管制**：对镓、锗、锑等关键矿产实施出口管制（2023-2025年陆续推出），稀土加工技术禁止出口
3. **稀土永磁体出口管控**：对钐钴磁铁、钕铁硼磁铁等实施出口许可证管理
4. **产业整合**：中国北方稀土、中国南方稀土集团等形成寡头格局
5. **战略储备**：建立国家稀土战略储备制度

这些政策使稀土从单纯的商品变成了地缘政治筹码。

### 1.6 全球稀土供应链"去中国化"努力

面对中国的主导地位，西方国家近年来积极推动供应链多元化：

- **美国**：重启Mountain Pass矿，推动国内加工产能建设，发布关键矿产战略
- **澳大利亚**：Lynas公司在马来西亚和澳大利亚建设加工厂
- **欧盟**：《关键原材料法案》（Critical Raw Materials Act），设定2030年本土加工目标
- **日本**：深海稀土泥开发研究，与澳大利亚、加拿大合作
- **"矿产安全伙伴关系"（MSP）**：美国主导的多边矿产供应链合作框架

但现实是：建立完整的稀土加工链需要5-10年时间和数十亿美元投资，短期内中国的主导地位难以撼动。2026年美伊战争的爆发进一步证明了这一点——当全球供应链因地缘冲突动荡时，掌握关键资源加工能力的国家拥有更大的战略主动权。

## 二、伊朗国家概况与现状

### 2.1 基本国情

| 项目 | 内容 |
|------|------|
| 国名 | 伊朗伊斯兰共和国（Islamic Republic of Iran） |
| 面积 | 164.8万平方公里（世界第17位） |
| 人口 | 约8,900万（2025年估计） |
| 首都 | 德黑兰（Tehran） |
| 政体 | 伊斯兰共和制（最高领袖+总统双元体制） |
| 最高领袖 | 阿里·哈梅内伊（已于2026年2月28日遇袭身亡）→ 穆杰塔巴·哈梅内伊继任 |
| GDP | 约4,000-4,500亿美元（PPP，2024年估计） |
| 主要产业 | 石油天然气、石化、汽车制造、矿业、农业 |
| 货币 | 伊朗里亚尔（IRR），实际流通常用土曼（Toman） |

### 2.2 政治体制

伊朗实行独特的"二元政治体制"：

- **最高领袖**：掌握最终决策权，控制军队（含伊斯兰革命卫队IRGC）、司法、外交大方向
- **总统**：负责日常行政，由选举产生，任期4年
- **议会（伊斯兰协商会议）**：立法机构，290个席位
- **宪法监护委员会**：审查候选人资格和法案合宪性
- **确定国家利益委员会**：最高领袖与议会之间的协调机构

**政治派系**：
- **保守派/强硬派**：主张抵抗西方压力，强调自力更生，支持"抵抗经济"
- **改革派/温和派**：主张与国际社会接触，推动经济开放
- 近年来保守派占据主导地位，改革派空间被大幅压缩

> **2026年战争影响**：哈梅内伊遇袭身亡后，伊朗经历了临时领导委员会的过渡阶段，随后由其子穆杰塔巴·哈梅内伊（据报受伤）继任最高领袖。权力交接的动荡加剧了国内的政治不确定性。

### 2.3 经济现状

#### 2.3.1 经济结构

伊朗经济高度依赖油气资源，但制裁严重制约了其经济潜力：

- **石油部门**：占GDP约15-20%，占出口收入约50-60%
- **制造业**：汽车（中东最大汽车生产国之一）、石化、钢铁、水泥
- **农业**：开心果、藏红花、水果为重要出口农产品
- **服务业**：约占GDP 50%以上
- **矿业**：铜、铁、锌、铬等，稀土开发处于早期阶段

#### 2.3.2 制裁下的经济困境

| 指标 | 战前状况（截至2025年底） |
|------|----------------------|
| 通货膨胀 | 长期保持在30-50%（2025年超过40%） |
| 货币贬值 | 里亚尔对美元从2018年约42,000贬至2025年约600,000+（非官方汇率） |
| 失业率 | 官方约10-12%，青年失业率估计25-30% |
| 石油出口 | 制裁前约250万桶/日，制裁后降至100-150万桶/日 |
| 外国投资 | 因制裁几乎停滞，西方企业大规模撤离 |
| 人才外流 | 大量受教育人口移民海外 |

#### 2.3.3 "抵抗经济"政策

面对制裁，伊朗提出"抵抗经济"（Resistance Economy）战略：

- 减少对石油出口的依赖
- 发展国内制造业和知识型经济
- 寻求非美元贸易结算（与中国用人民币结算石油贸易）
- 开发矿产资源作为替代出口收入来源
- 加强与中国、俄罗斯等非西方国家的经济联系

### 2.4 外交格局

| 国家/地区 | 关系状态 | 关键议题 |
|----------|---------|---------|
| 美国 | 敌对（2026年爆发战争） | 核问题、制裁、代理人冲突 |
| 中国 | 战略伙伴 | 25年全面合作协议、石油贸易、基建投资 |
| 俄罗斯 | 准盟友 | 军事合作（无人机）、能源合作 |
| 以色列 | 长期敌对（2026年联合美国打击伊朗） | 核威胁、代理人战争 |
| 沙特阿拉伯 | 2023年复交但结构性矛盾未解 | 地区主导权竞争 |
| 欧盟 | 复杂 | 核协议遗产，人权争议 |

### 2.5 军事与安全

- **伊斯兰革命卫队（IRGC）**：伊朗最强大的军事和经济力量
- **核计划**：铀浓缩纯度已达约60%，接近武器级别（90%），这是2026年战争的核心导火索
- **弹道导弹计划**：拥有中东最大的弹道导弹库
- **无人机技术**：向俄罗斯提供"沙赫德"系列无人机，展示了技术能力
- **"代理人网络"**：真主党、胡塞武装、伊拉克什叶派民兵等

### 2.6 社会与人口

- **2025-2026年大规模抗议**：延续2022年"玛莎·阿米尼"运动的社会不满
- **2026年伊朗屠杀事件**：政府对抗议者暴力镇压，加剧内部危机
- **库尔德分离主义危机**：2026年库尔德-伊朗危机升级
- 上述内部动荡成为美以判断军事行动时机的重要因素

## 三、伊朗矿产与稀土资源

### 3.1 伊朗矿产资源概况

伊朗位于阿尔卑斯-喜马拉雅造山带上，地质条件优越，矿产资源丰富：

| 矿产 | 全球排名/储量 | 备注 |
|------|-------------|------|
| 铜 | 全球前10 | 萨尔切什梅（Sar Cheshmeh）为世界级铜矿 |
| 铁矿石 | 储量约47亿吨 | 中东最大铁矿石生产国之一 |
| 锌 | 全球前15 | 安古兰（Angouran）矿为世界级锌铅矿 |
| 铬铁矿 | 中东主要生产国 | |
| 铀矿 | 自主开采 | 用于核燃料 |
| 稀土 | 探明储量有限，潜力待评估 | 处于早期勘探阶段 |

伊朗已探明的矿种超过68种，矿产地超过5,700处。矿业在GDP中占比约为1-2%，远低于其潜力。

### 3.2 伊朗稀土资源

#### 3.2.1 储量与分布

伊朗稀土资源勘探程度较低，已知的主要稀土矿化区域包括：

- **赛甘德（Saghand）地区**：位于亚兹德省，与铀矿伴生的稀土矿化
- **塞尔达什特（Chadormalu）地区**：铁矿中伴生稀土
- **西北部阿塞拜疆省**：部分区域发现稀土矿化迹象
- **呼罗珊省**：部分区域有稀土矿化报告

USGS将伊朗归入"其他国家"类别，未单独列出储量，说明目前尚缺乏可靠的大规模储量数据。

#### 3.2.2 开发现状

伊朗的稀土开发处于早期阶段：

- 伊朗地质调查局（GSI）和矿业发展组织（IMIDRO）一直在推动勘探
- 已有初步的实验室规模分离研究
- 缺乏商业化规模的开采和加工设施
- 主要技术瓶颈在于稀土分离冶炼技术
- 制裁导致难以获取西方技术和设备

#### 3.2.3 伊朗发展稀土的战略动机

1. **经济多元化**：减少对石油收入的依赖
2. **制裁规避**：矿产品出口受制裁限制较少
3. **吸引外资**：稀土作为战略资源可吸引中国等国投资
4. **国际议价**：掌握稀土资源可增加地缘政治谈判筹码
5. **产业升级**：为下游高科技产业提供原材料基础

> **战争影响**：2026年战争使上述所有战略动机在短期内失去可操作性。伊朗基础设施遭受严重打击（截至3月6日已有超过4,000栋民用建筑受损），矿业开发所需的基本条件——稳定的政治环境、基础设施、外资——均被严重破坏。

## 四、中伊关系与战略合作

### 4.1 中伊关系历史脉络

| 时期 | 关系发展 |
|------|---------|
| 1971年 | 伊朗（巴列维王朝）与中华人民共和国建交 |
| 1979年 | 伊朗伊斯兰革命，中国承认新政权 |
| 1980年代 | 两伊战争期间，中国向伊朗出售军事装备 |
| 1990年代 | 能源合作逐步加深，中国开始大量进口伊朗石油 |
| 2000年代 | 中国成为伊朗最大贸易伙伴之一 |
| 2016年 | 习近平访问伊朗，提升为全面战略伙伴关系 |
| 2021年 | 签署25年全面合作协议 |
| 2023年 | 中国斡旋沙特-伊朗恢复外交关系 |
| 2026年 | 美以对伊朗开战，中伊合作面临严峻考验 |

### 4.2 中伊25年全面合作协议

2021年3月签署的《中伊25年全面合作计划》涵盖以下领域：

- **能源**：中国对伊朗油气领域投资、石油贸易人民币结算
- **基础设施**：铁路、公路、港口建设（对接"一带一路"）
- **矿产资源**：联合勘探和开发，稀土等关键矿产合作可能包含在内
- **军事与安全**：技术合作、情报共享
- **金融**：推动去美元化贸易

需要注意：该协议更多是框架性文件，实际执行受中国企业对美国次级制裁的担忧制约，许多项目停留在意向阶段。石油贸易是合作中最活跃的部分。

### 4.3 中伊贸易关系

| 年份 | 双边贸易额（估计） | 特点 |
|------|------------------|------|
| 2018年 | 约350亿美元 | 以伊朗石油出口为主 |
| 2020年 | 约140亿美元 | 疫情+制裁双重打击 |
| 2022年 | 约160亿美元 | 逐步恢复 |
| 2024年 | 约200-250亿美元 | 灰色渠道活跃 |
| 2026年（战后） | 大幅萎缩（待观察） | 战争导致贸易中断 |

### 4.4 "一带一路"框架下的中伊合作

伊朗在"一带一路"中占据重要地理位置：

- **地理枢纽**：连接中亚、南亚和中东的关键节点
- **交通走廊**：中国-中亚-西亚经济走廊途经伊朗
- **港口**：恰巴哈尔港和阿巴斯港是重要的海运节点

然而制裁已使进展有限，2026年战争进一步使伊朗通道中断（详见第八章）。

## 五、制裁背景与地缘博弈

### 5.1 美国对伊朗制裁体系

| 时间 | 事件 |
|------|------|
| 1979年 | 美国大使馆人质危机后，首次冻结伊朗资产 |
| 1996年 | 《伊朗-利比亚制裁法案》 |
| 2010年 | 联合国安理会第1929号决议，多边制裁 |
| 2012年 | 美国对伊朗央行和石油出口实施全面制裁 |
| 2015年 | 伊核协议（JCPOA）签署，部分制裁解除 |
| 2018年 | 特朗普第一任期退出JCPOA，"极限施压" |
| 2025年 | 特朗普第二任期重启并升级"极限施压" |
| 2026年 | 从制裁升级为直接军事打击 |

### 5.2 制裁的灰色地带与规避

尽管制裁严厉，中伊贸易（尤其是石油）从未完全中断：

- **影子油轮船队**：关闭AIS系统、船对船转运
- **中间商网络**：通过阿联酋、马来西亚等地中转
- **人民币结算**：绕过美元体系
- **易货贸易**：石油换商品的直接交换

### 5.3 稀土在地缘博弈中的角色

稀土已成为中美战略竞争的重要维度。在这个背景下，伊朗作为潜在的稀土资源国地位微妙：

- 西方国家因制裁无法参与伊朗稀土开发
- 中国成为伊朗发展稀土产业最可能的合作方
- 这可能进一步强化中国在全球稀土供应链中的控制力
- 但2026年战争打乱了所有既定计划

## 六、中伊稀土合作前景与风险分析（战前评估）

> **注**：本章分析基于2026年2月战争爆发前的判断，战争对合作前景的颠覆性影响见第七、八章。

### 6.1 合作的潜在模式

| 模式 | 描述 | 战前可能性 |
|------|------|-----------|
| 技术换资源 | 中国提供稀土勘探/加工技术，换取矿产资源 | 高 |
| 合资企业 | 成立中伊合资矿业公司 | 中 |
| BOT模式 | 中国企业建设-运营-移交稀土加工厂 | 中 |
| 纯技术服务 | 中国企业提供地质勘探、技术咨询 | 高 |
| 全产业链投资 | 中国企业投资从开采到加工的全链条 | 低 |

### 6.2 合作的推动因素

1. **互补性强**：伊朗有资源，中国有技术和资本
2. **战略利益一致**：两国都有对抗美国单边主义的需求
3. **25年合作协议**：提供了宏观政策框架
4. **中国供应链安全**：寻求海外稀土资源多元化
5. **伊朗经济需求**：急需外资和技术发展矿业

### 6.3 战前情景分析

- **情景一（有限合作，最可能）**：合作停留在勘探和小规模试验阶段
- **情景二（深化合作）**：触发条件——制裁松动或中美关系极度恶化
- **情景三（战略突破）**：触发条件——伊朗发现大规模稀土矿床+地缘格局剧变

> **现实发展**：2026年2月28日战争爆发，使得情景一中的有限合作也被冻结。地缘格局确实发生了剧变，但方向与情景三预设的完全不同。

## 七、2026年美伊战争：爆发原因与经过

### 7.1 事件概述

**2026年2月28日**，美国和以色列联合发动了对伊朗的大规模军事打击：

| 项目 | 内容 |
|------|------|
| 发起时间 | 2026年2月28日 |
| 美方代号 | "Operation Epic Fury"（史诗之怒行动） |
| 以方代号 | "Roaring Lion"（咆哮雄狮） |
| 打击目标 | 核设施、军事设施、领导层（含政权更迭目标） |
| 重大结果 | 最高领袖哈梅内伊被暗杀 |
| 截至3月10日 | 冲突仍在持续（已进行1周零3天） |

### 7.2 战争爆发的六大原因

#### 原因一：伊朗核计划逼近武器级别（直接导火索）

| 时间 | 核计划进展 |
|------|-----------|
| 2015年 | JCPOA签署，限制铀浓缩至3.67% |
| 2018年 | 美国退出协议后，伊朗逐步突破限制 |
| 2024年 | 铀浓缩纯度达约60%（武器级为90%） |
| 2026年初 | 最后一轮核谈判（日内瓦）破裂 |

美以判断外交手段已无法阻止伊朗获得核武器，选择了军事手段。

#### 原因二：2023年以来中东危机升级链

```
2023.10 哈马斯袭击以色列 → 加沙战争
   ↓
真主党-以色列边境持续交火
   ↓
2024年 伊朗与以色列直接冲突（"十二日战争"）
   ↓
胡塞武装攻击红海航运（红海危机）
   ↓
伊朗支持的民兵攻击美军基地
   ↓
2025-2026年 核谈判破裂，局势总爆发
   ↓
2026.2.28 美以联合打击伊朗
```

#### 原因三：伊朗内部危机创造了"时机窗口"

- **2025-2026年大规模抗议**：经济困境引发社会动荡
- **严重经济危机**：通胀超40%，货币崩溃
- **2026年伊朗屠杀事件**：暴力镇压抗议者
- **库尔德分离主义危机**：加剧国内不稳定

美以判断伊朗政权处于内外交困的脆弱期。

#### 原因四：特朗普第二任期的"极限施压"升级

特朗普政府的目标从核遏制升级为**政权更迭**（regime change），与以色列内塔尼亚胡政府的战略利益高度一致。

#### 原因五：以色列的存在性安全计算

以色列将伊朗核武器视为存在性威胁：伊朗长期否认以色列存在权，并资助多个针对以色列的武装力量。

#### 原因六：伊朗"代理人网络"的持续威胁

| 代理人 | 对美以的威胁 |
|--------|------------|
| 真主党 | 与以色列持续交火 |
| 胡塞武装 | 攻击红海商船和以色列目标 |
| 伊拉克什叶派民兵 | 攻击美军基地 |
| 哈马斯 | 加沙冲突持续 |

美以认为只有打击伊朗本体，才能从根源上瓦解代理人网络。

### 7.3 战争经过（截至3月10日）

#### 7.3.1 军事行动

- 美以对伊朗军事设施、核设施、领导层进行大规模空袭
- 美国潜艇在印度洋（斯里兰卡南部）击沉伊朗军舰
- 超过50艘伊朗海军舰艇被击沉
- 多名伊朗高级军事指挥官被击杀

#### 7.3.2 伊朗的反击

- 向以色列发射弹道导弹
- 攻击美军在卡塔尔、阿联酋、巴林的基地
- **关闭霍尔木兹海峡**——对全球经济影响最大的举措
- 攻击沙特Aramco炼油厂、迪拜机场等海湾基础设施
- 真主党重新攻击以色列，开辟黎巴嫩战线

#### 7.3.3 伤亡情况

| 方面 | 伤亡 |
|------|------|
| 伊朗（伊方称） | 1,255-1,675人死亡 |
| 伊朗（美以称） | 约3,000名军人死亡 |

### 7.4 霍尔木兹海峡危机

这是战争中影响全球经济最深远的事件：

| 时间 | 事态发展 |
|------|---------|
| 2月28日 | IRGC通过VHF无线电警告所有船只禁止通过海峡 |
| 3月1日 | 多艘油轮在海峡附近遭袭，3名印度船员死亡 |

- 伊拉克南部油田产量下降**70%**

## 八、战争对中国的全面影响

### 8.1 能源安全：最直接、最严重的冲击

#### 8.1.1 霍尔木兹海峡——中国能源命脉受威胁

中国是通过霍尔木兹海峡石油运输的最大单一接收国：

| 指标 | 数据 |
|------|------|
| 中国日均原油进口 | 约1,000万桶 |
| 来自中东的占比 | 约40-50% |
| 海峡通行石油的亚洲占比 | 75%（中日韩印合计） |
| 海峡通行LNG的亚洲占比 | 约80% |

霍尔木兹海峡事实上关闭后，中国面临的直接冲击包括：

- **油价飙升**：布伦特原油从约70美元涨至100美元以上，油价每涨10美元，中国年增进口成本约**365亿美元**
- **LNG价格翻倍**：亚洲LNG现货价格从约12美元飙升至25.40美元/MMBtu
- **卡塔尔停产**：全球20%LNG产能中断，中国是卡塔尔LNG主要买家之一
- **伊拉克减产**：南部油田产量下降70%，伊拉克是中国第三大石油来源国

#### 8.1.2 中国的"特权通道"

一个关键细节：**伊朗对中国船只网开一面**。

- 3月4日：报道称伊朗只允许中国船只通过海峡
- 3月5日：散货船 *Iron Maiden*（上海船东）广播"CHINA OWNER"后成功通过
- 3月5日：IRGC宣布海峡只对美以及其西方盟友关闭
- 3月7日：*Sino Ocean*（中国船东）广播中国身份后成功通过

**这意味着**：

- 中国在短期内有一定的通行优势，体现了中伊关系的特殊性
- 但这种特权是脆弱的——依赖伊朗单方面决定
- 更大的问题是上游生产国的停产：即使中国船能通过，沙特、卡塔尔、科威特、伊拉克等纷纷停产或减产，能买到的石油本身就大幅减少

| 航空业 | 中东空域大面积关闭，中欧航线受影响，航油成本飙升 |
| 金融市场 | 全球避险情绪升温，A股/港股承压，人民币面临贬值压力 |

### 8.3 中伊关系与合作受到的冲击

#### 8.3.1 石油贸易中断

- 中国是伊朗石油最大买家（战前估计100万桶/日以上）
- 伊朗基础设施在战争中遭到大规模破坏
- 即使战争结束，伊朗石油产能恢复需要数月甚至数年

#### 8.3.2 25年合作协议面临严峻挑战

- 哈梅内伊遇袭身亡后，伊朗权力交接动荡
- 政局不确定性急剧增加
- 中国在伊朗的投资项目面临战争破坏和政治风险
- 25年协议的执行前景变得高度不确定

#### 8.3.3 对中伊稀土合作的具体影响

| 战前预期 | 战后现实 |
|---------|---------|
| 有限但逐步推进的勘探合作 | 所有合作项目冻结 |
| 稀土勘探技术交流 | 伊朗矿业基础设施可能遭受破坏 |
| 长期合资开发的可能性 | 伊朗政治前景高度不确定 |
| 中国技术换资源模式 | 伊朗短期内无力推进任何矿业项目 |

### 8.4 地缘政治影响

#### 8.4.1 中美关系进一步承压

- 美国对伊动武直接损害了中国最重要的中东经济伙伴
- 中国警告战争可能外溢至高加索地区并蔓延到中东以外
- 中国面临艰难的外交平衡：批评美以单边行动 vs 避免与美国全面对抗
- 稀土出口管制可能成为中国的反制工具之一

#### 8.4.2 "一带一路"受阻

- 伊朗是中国-中亚-西亚经济走廊的关键节点，战争使该通道中断
- 中亚国家通过伊朗港口的南向出海通道被切断
- C6国家（阿塞拜疆、哈萨克斯坦、吉尔吉斯斯坦、塔吉克斯坦、土库曼斯坦、乌兹别克斯坦）3月3日紧急磋商应对方案
- 恰巴哈尔港等项目前景不明

#### 8.4.3 中国在中东的斡旋角色受挫

- 2023年中国成功斡旋沙伊复交，外交声望大增
- 此次美以对伊动武，中国未能预防或阻止
- 沙特、阿联酋等海湾国家遭伊朗报复性打击，中国此前与各方维持的平衡关系面临考验

### 8.5 中国的应对策略

#### 8.5.1 短期应急

| 措施 | 说明 |
|------|------|
| 释放战略石油储备 | 中国战略石油储备约10-15亿桶，可提供约3个月缓冲 |
| 加速非中东来源进口 | 俄罗斯管道油气、中亚、西非、巴西、委内瑞拉 |
| 利用"特权通道" | 争取通过霍尔木兹海峡的持续通行权 |
| 稳定国内市场 | 控制成品油价格，补贴受影响行业 |

#### 8.5.2 中长期战略调整

| 策略 | 说明 |
|------|------|
| 能源来源多元化 | 减少对中东依赖，加强俄罗斯、中亚能源合作 |
| 新能源加速 | 此次危机加速太阳能、风电、核电、储能发展 |
| 战略储备扩建 | 扩大石油和LNG战略储备容量 |
| 稀土反制筹码 | 在中美博弈中更积极使用稀土出口管制工具 |
| 中东外交重塑 | 战后可能在伊朗重建中争取更大角色 |
| 海上通道安全 | 加强海军在印度洋的存在，保护能源运输线 |
| 人民币国际化 | 利用美元武器化引发的不满，推动能源贸易用人民币结算 |

### 8.6 危中有机：潜在的战略机遇

尽管短期冲击巨大，危机也为中国创造了一些机遇：

1. **战后伊朗重建**：如果战争结束，中国作为伊朗最大经济伙伴可能获得优先参与重建的机会
2. **能源转型加速**：化石能源供应危机倒逼中国更快摆脱石油依赖，加速新能源产业发展
3. **全球南方团结**：美国单边军事行动在发展中国家引发广泛批评，中国可借此巩固在全球南方的影响力
4. **稀土战略价值凸显**：全球供应链动荡进一步证明了关键矿产主导权的战略价值，中国的稀土筹码更加重要
5. **人民币国际化**：美元武器化和中东动荡可能推动更多国家接受人民币贸易结算

### 8.7 影响总评

| 维度 | 短期影响 | 中长期影响 |
|------|---------|-----------|
| 能源安全 | 严重——油价飙升，供应链中断 | 加速能源多元化和新能源转型 |
| 经济增长 | 负面——成本上升，市场波动 | 取决于战争持续时间和能源价格 |
| 中伊合作 | 冻结——所有项目暂停 | 战后重建可能带来新机遇 |
| 稀土产业 | 中伊稀土合作推迟 | 中国稀土战略地位进一步强化 |
| 地缘政治 | 中美关系承压，中东布局受挫 | 中国在中东角色可能重新定义 |
| 一带一路 | 伊朗通道中断 | 需要替代路线规划 |
| 金融市场 | 剧烈波动 | 人民币国际化机遇 |

## 总结与研究建议

### 核心结论

1. **中国稀土的全球主导地位短期内不可替代**，2026年战争进一步凸显了关键矿产控制权的战略价值。

2. **伊朗稀土资源具有潜力但远未成熟**，战争使其本就薄弱的矿业开发条件遭受进一步破坏。

3. **中伊稀土合作在短期内被冻结**，25年合作协议的矿产合作条款因战争失去可操作性，恢复时间取决于战争结束后的政治重建进度。

4. **能源安全是战争对中国最直接的冲击**。霍尔木兹海峡危机暴露了中国对中东能源通道的高度依赖，尽管伊朗给予中国船只"特权通行"，但上游产油国停产的影响更为根本。

5. **危机也蕴含机遇**。此次战争可能加速中国在能源多元化、新能源转型、稀土战略工具化、人民币国际化等方面的进程，并在战后伊朗重建中争取更大角色。

6. **全球稀土博弈的格局因战争而强化而非改变**。中国的稀土主导地位在全球供应链动荡中反而更加重要，西方"去中国化"的稀土供应链建设因资源分散、成本高昂、地缘风险而进展缓慢。
