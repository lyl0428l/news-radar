# 新闻爬虫完整时序图 - 学习版

## 第一步：最简单的版本（只有2个参与者）

```mermaid
sequenceDiagram
    participant S as 调度器
    participant M as 主程序
    
    S->>M: 每小时触发爬取
    M-->>S: 爬取完成
```

## 第二步：添加爬虫（3个参与者）

```mermaid
sequenceDiagram
    participant S as 调度器
    participant M as 主程序
    participant C as 爬虫
    
    S->>M: 1. 触发爬取
    M->>C: 2. 爬取新浪
    C-->>M: 3. 返回10条新闻
    M->>C: 4. 爬取腾讯
    C-->>M: 5. 返回10条新闻
    M-->>S: 6. 爬取完成
```

## 第三步：添加并发（用循环表示多个站点）

```mermaid
sequenceDiagram
    participant S as 调度器
    participant M as 主程序
    participant C as 爬虫
    
    S->>M: 1. 触发爬取
    
    loop 每个站点（10个）
        M->>C: 2. 爬取站点
        C-->>M: 3. 返回10条新闻
    end
    
    M-->>S: 4. 爬取完成
```

## 第四步：添加详情页抓取

```mermaid
sequenceDiagram
    participant S as 调度器
    participant M as 主程序
    participant C as 爬虫
    participant W as 网站
    
    S->>M: 1. 触发爬取
    
    loop 每个站点（10个）
        M->>C: 2. 爬取列表页
        C->>W: 3. HTTP请求
        W-->>C: 4. 返回HTML
        C-->>M: 5. 返回10条新闻标题
        
        loop 每条新闻（10条）
            M->>C: 6. 抓取详情页
            C->>W: 7. HTTP请求详情
            W-->>C: 8. 返回详情HTML
            C-->>M: 9. 返回正文/图片
        end
    end
    
    M-->>S: 10. 爬取完成
```

## 第六步：添加数据存储

```mermaid
sequenceDiagram

    participant C as 爬虫
    participant W as 网站

        C-->>M: 5. 返回10条新闻
        
        loop 每条新闻（10条）
            M->>C: 6. 抓取详情页
            C->>W: 7. HTTP请求详情
            W-->>C: 8. 返回详情HTML
            C-->>M: 9. 返回正文/图片URL
            
            loop 每张图片（最多20张）
                M->>MS: 10. 下载图片
                MS->>W: 11. HTTP请求图片
                W-->>MS: 12. 返回图片数据
                MS-->>M: 13. 返回本地路径
            end
        end
    end
```

## 第七步：添加微信推送（完整版）

```mermaid
sequenceDiagram
    participant S as 调度器
    participant M as 主程序
    participant C as 爬虫
    participant W as 网站
    participant MS as 媒体存储
    participant ST as 存储
    participant DB as 数据库
    participant N as 通知
    participant WX as 微信
    
    S->>M: 1. 触发爬取
    
    loop 每个站点（10个）
        M->>C: 2. 爬取列表页
        C->>W: 3. HTTP请求
        W-->>C: 4. 返回HTML
        C-->>M: 5. 返回10条新闻
        
        loop 每条新闻（10条）
            M->>C: 6. 抓取详情页
            C->>W: 7. HTTP请求详情
            W-->>C: 8. 返回详情HTML
            C-->>M: 9. 返回正文/图片URL
            
            loop 每张图片（最多20张）
                M->>MS: 10. 下载图片
                MS->>W: 11. HTTP请求图片
                W-->>MS: 12. 返回图片数据
                MS-->>M: 13. 返回本地路径
            end
        end
    end
    
    M->>ST: 14. 保存到数据库
    ST->>DB: 15. INSERT OR IGNORE
    DB-->>ST: 16. 返回新增条数
    ST-->>M: 17. 保存完成
    
    M->>ST: 18. 保存JSON文件
    ST-->>M: 19. 返回文件路径
    
    M->>N: 20. 检查重大新闻
    N->>WX: 21. 推送重大新闻
    WX-->>N: 22. 推送成功
    
    M->>N: 23. 发送汇总
    N->>WX: 24. 推送汇总
    WX-->>N: 25. 推送成功
    
    M-->>S: 26. 爬取完成
```

## 学习建议

1. **从第一步开始画**，不要直接画完整版
2. **每画一步就运行**，检查语法是否正确
3. **逐步添加参与者**，理解每个参与者的作用
4. **最后添加细节**（循环、条件、注释）
