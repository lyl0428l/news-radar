第1步最简单的版本（2个参与者）

学习目标：掌握基本语法
participant 定义参与者，->> 调用，-->> 返回
sequenceDiagram participant S as 调度器 participant M as 主程序 S->>M: 每小时触发爬取 M-->>S: 爬取完成

📝 代码解释

• participant S as 调度器 - 定义第一个参与者
• S->>M: 消息 - 实线箭头表示"调用"
• M-->>S: 消息 - 虚线箭头表示"返回"

第2步添加爬虫（3个参与者）

学习目标：添加多个参与者
爬虫需要访问网站获取数据
sequenceDiagram participant S as 调度器 participant M as 主程序 participant C as 爬虫 S->>M: 1. 触发爬取 M->>C: 2. 爬取新浪 C-->>M: 3. 返回10条新闻 M->>C: 4. 爬取腾讯 C-->>M: 5. 返回10条新闻 M-->>S: 6. 爬取完成

📝 代码解释

• 新增 C as 爬虫 参与者
• 主程序调用爬虫两次（新浪、腾讯）
• 每次调用返回10条新闻

第3步添加并发（用循环表示多个站点）

学习目标：使用 loop 循环
实际有10个站点，用循环简化表示
sequenceDiagram participant S as 调度器 participant M as 主程序 participant C as 爬虫 S->>M: 1. 触发爬取 loop 每个站点（10个） M->>C: 2. 爬取站点 C-->>M: 3. 返回10条新闻 end M-->>S: 4. 爬取完成

📝 代码解释

• loop 循环条件 - 开始循环
• end - 结束循环
• 循环内的操作会重复执行10次

第4步添加详情页抓取

学习目标：添加外部系统（网站）
爬虫需要访问网站获取HTML，然后解析
sequenceDiagram participant S as 调度器 participant M as 主程序 participant C as 爬虫 participant W as 网站 S->>M: 1. 触发爬取 loop 每个站点（10个） M->>C: 2. 爬取列表页 C->>W: 3. HTTP请求 W-->>C: 4. 返回HTML C-->>M: 5. 返回10条新闻标题 loop 每条新闻（10条） M->>C: 6. 抓取详情页 C->>W: 7. HTTP请求详情 W-->>C: 8. 返回详情HTML C-->>M: 9. 返回正文/图片 end end M-->>S: 10. 爬取完成

📝 代码解释

• 新增 W as 网站 外部系统
• 嵌套循环：外层10个站点，内层10条新闻
• HTTP请求：爬虫请求网站，网站返回HTML

图片需要下载到本地保存
sequenceDiagram participant S as 调度器 participant M as 主程序 participant C as 爬虫 participant W as 网站 participant MS as 媒体存储 S->>M: 1. 触发爬取 loop 每个站点（10个） M->>C: 2. 爬取列表页 C->>W: 3. HTTP请求 W-->>C: 4. 返回HTML C-->>M: 5. 返回10条新闻 loop 每条新闻（10条） M->>C: 6. 抓取详情页 C->>W: 7. HTTP请求详情 W-->>C: 8. 返回详情HTML C-->>M: 9. 返回正文/图片URL loop 每张图片（最多20张） M->>MS: 10. 下载图片 MS->>W: 11. HTTP请求图片 W-->>MS: 12. 返回图片数据 MS-->>M: 13. 返回本地路径 end end end M-->>S: 14. 爬取完成

• 媒体存储负责下载图片并保存到本地
