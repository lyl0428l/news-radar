# 新闻聚合爬虫平台

自动爬取国内外 15 家新闻门户网站的 TOP 10 热点新闻，本地存储 + Web 界面浏览。

## 功能特性

- **15 个新闻源**: 新浪、网易、腾讯、搜狐、凤凰、新华、人民、央视、澎湃、界面 + CNN、BBC、Reuters、AP、NYT
- **TOP 10 热榜**: 每个站点只抓取热度排名前 10 的新闻，带  排名字段
- **定时爬取**: APScheduler 每小时自动执行，首次启动立即爬取
- **双重存储**: SQLite 数据库（URL 去重）+ JSON 归档（按日期/小时）
- **中文 Web UI**: Flask 本地界面，支持筛选、搜索、分页、排名徽章
- **数据导出**: 支持 CSV / JSON 导出（带 UTF-8 BOM 兼容 Excel）
- **爬取轮次**: 可按某次爬取时间点查看当时的新闻快照
- **手动爬取**: Web 页面一键触发爬取
- **自动刷新**: 前端每 5 分钟检测新数据并提示刷新
- **健康监控**: 爬虫状态面板，异常时首页显示警告横幅

## Web 界面功能

| 功能 | 说明 |
|------|------|
| 来源筛选 | 下拉框选择单个新闻源 |
| 语言筛选 | 中文 / 英文 |
| 时间范围 | datetime-local 精确到分钟 |
| 爬取轮次 | 按某轮爬取快照查看 |
| 关键词搜索 | 全文搜索（标题+摘要+正文） |

| 手动爬取 | 一键触发爬取，完成后自动刷新 |

| 爬虫监控 | /health 页面查看各站点状态 |

## 配置说明

编辑  修改：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
|  | 1 | 爬取间隔（小时） |
|  | 15 | 请求超时（秒） |
|  | 5 | 并发线程数 |
|  | 5000 | Web 服务端口 |
|  | 30 | 数据保留天数 |

## 技术栈

- Python 3.10+
- Flask (Web)
- APScheduler (定时任务)
- Requests + BeautifulSoup4 + lxml (爬取解析)
- feedparser (RSS 解析)
- SQLite (存储)
